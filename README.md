# Portfolio de Valentín González

Mi portfolio personal como Software Engineer con 3 años de experiencia. Construido con Astro, Tailwind CSS e inspirado en el estilo limpio y futurista de [luffy.sh](https://luffy.sh).

## Características

- ✨ Diseño limpio y minimalista
- 🎨 Tema futurista con scroll suave
- 📱 Completamente responsive
- ⚡ Rendimiento optimizado (Astro)
- 🎭 Tema Shanks (One Piece)
- 🚀 Desplegado en GitHub Pages

## Secciones

- **Hero**: Presentación principal
- **About**: Información profesional y habilidades
- **Experience**: Timeline de experiencia laboral
- **Projects**: Proyectos personales destacados
- **Contact**: Formas de contacto

## Tech Stack

- [Astro](https://astro.build) - Framework web moderno
- [Tailwind CSS](https://tailwindcss.com) - Utilidades CSS
- [GitHub Pages](https://pages.github.com) - Hosting

## Experiencia Laboral

## Instalación Local

```bash
# Clonar repositorio
git clone https://github.com/valentin-gonzalez/portfolio.git
cd portfolio

# Instalar dependencias
npm install

# Ejecutar servidor de desarrollo
npm run dev

# Hacer build para producción
npm run build
```

## Despliegue

El portfolio se despliega automáticamente en GitHub Pages cuando se hace push a la rama main.

```bash
npm run build
```
